Question-1: What are the different ways to select an element in the DOM?

Ways to Select Elements in the DOM:
  1. getElementById(): Selects an element by its unique ID.
  2. getElementsByClassName(): Selects elements by a specific class name. It is used to find multiple elements.
  3. getElementsByTagName(): Selects elements by a specific tag name.
  4. querySelector(): Selects the first matching element using a CSS selector. It only selects the first matching element.
  5. querySelectorAll(): Selects all matching elements using a CSS selector (returns a NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML returns the entire HTML content, including tags.

InnerText only returns visible text (ignoring hidden elements).

TextContent returns all text inside the element, including hidden content but without HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single listener is added to the parent, improving performance. This is useful for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event first triggers on a specific target element and then propagates upward to its parent elements. It works by default, making it useful for event delegation. The bubbling can be stopped using 'event.stopPropagation()'.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element
const newElement = document.createElement('div');
Add the element to the DOM
parentElement.appendChild(newElement);
Remove the element from the DOM
parentElement.removeChild(newElement);
